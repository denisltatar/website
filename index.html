RaceIt

Type of Work: Game Development

Number of Hours: 20+ hours

Software Used:

Languages: Java

Github Repository:

RaceIt was a highschool project that was created as a summative for my grade 12 computer science course (June, 2018). I developed this two player racing game which features two tracks along with the choice of four different cars. Split screen, pause, car boost, lap count are all features implemented into the game and in-game physics are applied using libGDX game development freamwork. Each track features grassy or sandy areas that reduces car speed by 60%. Each track also features its own checkpoints players must cross to complete a lap.
